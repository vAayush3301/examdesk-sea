# ExamDesk (Android)

A secure, fullscreen Computer-Based Testing (CBT) app for Android, designed to conduct exams with minimal distraction.

## Features

* Fullscreen exam mode
* Timer with auto-submit
* Secure test environment
* Clean and scrollable interface
* Firebase-powered notifications and updates

## Installation

* Install the APK on your Android device
* Launch the app and proceed with login

## Usage

* Select and attempt available tests
* Timer runs automatically during exams
* App maintains focus to reduce distractions

## Development

* Built with Java and Android SDK
* Uses Firebase (FCM, Crashlytics)
* Material UI components
